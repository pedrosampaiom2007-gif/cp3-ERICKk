## Descrição do Projeto

Este projeto consiste no desenvolvimento de um sistema simulador de caixa eletrônico (ATM) utilizando a linguagem C.

O sistema funciona via terminal/console e permite ao usuário realizar operações bancárias básicas através de um menu interativo.

O projeto foi desenvolvido utilizando conceitos estudados na disciplina, como:

- Estruturas de decisão (`switch-case`)
- Estruturas de repetição (`do-while`)
- Funções
- Modularização
- Validação de dados

---

## Funcionalidades Implementadas

- Consultar saldo
- Realizar saque
- Realizar depósito
- Menu interativo
- Validação de opções inválidas
- Validação de saldo insuficiente
- Validação de valores inválidos
